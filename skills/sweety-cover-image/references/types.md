# 类型构图指南

## 类型画廊

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| `hero` | 大视觉冲击力，标题叠加 | 产品发布、品牌推广、重大公告 |
| `conceptual` | 概念可视化，抽象核心思想 | 技术文章、方法论、架构设计 |
| `typography` | 以文字为主的布局，突出标题 | 观点文章、引用、见解 |
| `metaphor` | 视觉隐喻，以具象表达抽象 | 哲学、成长、个人发展 |
| `scene` | 氛围场景，叙事感 | 故事、旅行、生活方式 |
| `minimal` | 极简构图，大量留白 | 禅意、专注、核心概念 |

## 类型专属构图

| 类型 | 构图指南 |
|------|----------|
| `hero` | 大焦点视觉（60-70% 面积），标题覆盖在视觉上，戏剧性构图 |
| `conceptual` | 抽象形状代表核心概念，信息层次，干净区域 |
| `typography` | 标题为主要元素（40%+ 面积），最少辅助视觉，强层次 |
| `metaphor` | 具象物体/场景代表抽象思想，象征元素，情感共鸣 |
| `scene` | 氛围环境，叙事元素，渲染氛围的灯光和色彩 |
| `minimal` | 单一焦点元素，大量留白（60%+），仅保留必要形状 |
