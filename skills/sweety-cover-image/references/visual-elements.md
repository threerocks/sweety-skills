# 视觉元素库

| 二进制 / 数据流 | 数据、算法 |

## 渲染风格专属处理

| 渲染风格 | 元素风格 |
|----------|----------|
| `flat-vector` | 几何、简单形状、均匀填充 |
| `hand-drawn` | 素描、有机、涂鸦风 |
| `painterly` | 柔和边缘、笔触 |
| `digital` | 精确、渐变提示、精致 |
| `pixel` | 8-bit 粗像素、网格对齐 |
| `chalk` | 粉尘质感、纹理、黑板风 |
