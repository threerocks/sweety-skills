# 公众号爆款开头模板

这些模板用于文章前三句话或首段。使用时必须先确认标题、主题、目标读者和正文能兑现的内容；模板只能增强开场，不允许编造、夸大或悬空。

## 8 种黄金开头模板

| 模板 | 方法 | 使用要点 |
|------|------|----------|
| 问句式 | 用设问、反问或疑问句开头，制造悬念，引发思考 | 问题要直击读者心中的困惑，让人想继续看答案 |
| 对话式 | 把文章中最亮眼、最有爆点的一句话提前到开头 | 对话要短促、有悬念，最好来自正文真实场景 |
| 互动式 | 开篇直接用“你”与读者对话，像朋友聊天 | 多用“你有没有”“你是不是也”，但要指向明确人群 |
| 自我表达式 | 通过自黑、自嘲或特别经历开篇 | 要么有趣，要么特别，给读者一个关心作者的理由 |
| 题文呼应式 | 开头直接解释或补充标题关键词 | 标题留悬念或不易理解时，开头第一时间解题 |
| 名人名言式 | 引用或改造名言、俗语、诗句 | 高阶玩法是解构，但不要伪造名人真实说过 |
| 热点式 | 用热点人物、事件或影视作品切入 | 必须是真热点，且快速建立与正文的关联 |
| 粉丝投稿式 | 用真实来信、留言或提问作为引子 | 没有真实来源时不要写“投稿”，改成常见问题 |

## 3 种增强方法

| 方法 | 用法 | 注意 |
|------|------|------|
| 痛点描述法 | 开头直接抛出目标读者最具体的痛点问题 | 痛点越具体越好，避免泛泛而谈 |
| 故事描写法 | 用一个极具画面感的小故事开场 | 故事必须服务正文，不要为了画面感虚构关键事实 |
| 颠覆认知法 | 第一句话提出一个反常识判断 | 正文必须能证明这个判断，不能只靠口号 |

## 选择规则

1. 正文有强故事：优先对话式、故事描写法、自我表达式。
2. 正文是观点判断：优先问句式、颠覆认知法、题文呼应式。
3. 正文是方法清单：优先互动式、痛点描述法、题文呼应式。
4. 正文有真实读者材料：可以用粉丝投稿式，但要避免泄露隐私。
5. 正文与热点强相关：可以用热点式，但第一段内要交代和正文的关系。

## 三句话自检

- 如果我是目标读者，我会想继续往下看吗？
- 它是否承接了标题引发的情绪或好奇？
- 情绪是否一致、简洁有力、没有废话？
- 是否 3 句话内进入正题？
- 是否触动了读者的某个痛点或共鸣点？
- 是否让读者感觉“这就是我”？
- 是否承诺了正文真实能交付的内容？
