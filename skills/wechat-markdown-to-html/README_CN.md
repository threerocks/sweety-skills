# wechat-markdown-to-html

纯公众号文章排版 skill。

- Markdown 或纯文本 -> 微信公众号兼容的内联样式 HTML
- 30 套主题，5 大分类，可视化画廊选择
- 自动识别对话体、连续图片、金句，补成 `dialogue`、`gallery`、`callout`
- CJK 排版修复、脚注、图片复制、Obsidian `![[image]]` 兼容
- 可直接替换 `sweety-post-to-wechat` 的 Markdown 渲染入口

完整使用说明、主题清单、容器语法、测试命令和验收清单见 [references/guide.md](references/guide.md)。
